Format & Scoring

Match Format: This tournament has been designated to count for USTA Level 5 Ranking and uses the Flighted Draw Ranking Table. In the unlikely event there is only one flight, the standard table for a Compass Draw will be used.
Tiebreaks: All matches will be the best of 3 Tiebreak sets and will use Regular Scoring, except that in the unlikely event that 5 entries are received in a division, the best of 3 Tiebreak sets with a 10-Point Match Tiebreak will be played. Note that in the event of inclement weather the USTA Regulations permit the Referee to shorten the match format from those described below.
